# Web-Based Image Editor

## Description
This project is a web-based image editor built using HTML, CSS, and JavaScript.
It allows users to edit images directly in the browser without any external software.

## Features
- Adjust brightness, contrast, and saturation
- Apply grayscale, sepia, invert, and blur effects
- Hue rotation and opacity control
- Preset filters like Vintage, Old School, Drama, and Cinematic
- Real-time preview of image changes

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project was created to practice JavaScript DOM manipulation,
CSS filters, and basic image editing concepts.
